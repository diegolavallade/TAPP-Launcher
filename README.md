# TAPP Launcher

TAPP Launcher is a Windows desktop application for launching TAPP packages.

## Overview

- WPF-based launcher app
- Reads package manifests and starts TAPP packages
- Includes helper scripts for packaging

## Getting Started

1. Open `TAPP Launcher.csproj` in Visual Studio.
2. Restore NuGet packages if prompted.
3. Build and run the project.

## Scripts

- `tapp_pack.py`: Package helper for TAPP content
- `tapp_pack_win.py`: Windows packaging helper
